# Desafio-markdown
Desafio do curso da DIO Git e Github

## OBJETIVOS:

**💻 1. Criar uma conta no Github** 

    - [x] Adicionar seu README.MD ao criar o repositório
    - [ ] Utilize o Github.dev para editar o projeto
    - [ ] Repositorio precisa ser público

**📝 2. Colaboração**

    - [ ] Adicione um colaborador ao seu repositorio
    - [ ] Ambos devem fazer um clone do repositorio em suas maquinas locais.

**📝 3. Realizem alterações:**

    - [ ] COMMITS
    - [ ] ISSUES
    - [ ] PULL requests

**📝 4.  Formatação com Markdown:** 

    - [ ] Edite o arquivo "README.md" em seu repositório
    - [ ] Usar linguagem Markdown para formatar informações sobre o desafio.
