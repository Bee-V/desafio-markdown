# Desafio-markdown
Desafio do curso da DIO Git e Github

## OBJETIVOS:

**💻 1. Criar uma conta no Github** 

    - [x] Adicionar seu README.MD ao criar o repositório
    - [X] Utilize o Github.dev para editar o projeto
    - [x] Repositorio precisa ser público

**📝 2. Colaboração**

    - [x] Adicione um colaborador ao seu repositorio
    - [x] Ambos devem fazer um clone do repositorio em suas maquinas locais.

**📝 3. Realizem alterações:**

    - [x] COMMITS
    - [ ] ISSUES
    - [x] PULL requests

**📝 4.  Formatação com Markdown:** 

    - [x] Edite o arquivo "README.md" em seu repositório
    - [x] Usar linguagem Markdown para formatar informações sobre o desafio.
